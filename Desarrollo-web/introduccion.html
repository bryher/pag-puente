Introducción al Desarrollo Web

El Desarrollo Web consiste en la creación de sitios web. Esto se lo puede lograr mendiate el uso de ciertas tecnologías.



HTML y su significado


Mediante el uso de HTML, que es un lenguaje de Marcado de hipertexto, un página web se puede dividir en componentes y elementos que lo estructuran de manera ordenada. HTML le dice al Navegador que es cada parte del sitio web.

Es verdad que HTML organiza los elementos a los ojos del navegador, pero a los ojos del usuario esto no se ve visualmente ordenado, es por eso que existe otro lenguaje.



CSS y su significado


Luego de HTML surgió CSS, hoja de estilo en cascada, el cual tiene como función principal, dar un aspecto visual muy llamativo e impecable para el usuario. Este lenguaje de estilos, permite realizar hermosos diseños web, además de facilitar el trabajo para los desarrolladores.

Pero esto no es todo lo que existe en desarrollo web, pues HTML y CSS son netamente estáticos, no permiten que un sitio web sea interactivo con el usuario, es por eso que existe otro lenguaje.



JavaScript y su significado


JavaScript es un lenguaje de programación, este permite que el usuario realice eventos e interactúe con el DOM. Este lenguaje se ejecuta directamente en el navegador, y actuamente se ha convertido en un estándar para el desarrollo web.


HTML, CSS y JavaScript son las bases del desarrollo web, el inicio del camino de un futuro desarrollador. En este curso Aprenderás estas tres tecnologías, paso a paso y sin costo alguno. Mira el video sobre la introduccio a este curso.
